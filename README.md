# Data Structures and Algorithms

Coursework from a Data Structures and Algorithms course at the University of Wisconsin–Madison, implemented in Java. This repository contains a series of themed projects and exercises covering core data structures, algorithmic design, and object-oriented software design principles.

**Note:** This repository is a portfolio showcase of completed coursework, not a runnable pipeline. Code and outputs are provided for review.

## What's Inside

A collection of themed projects (e.g., `AirplaneBoardingSystem`, `AmongUs`, `BusStopTree`, `FrogGame`, `TicketRush`, `WorkoutBuilder`, and others) that apply data structures and algorithms concepts to simulated real-world scenarios, alongside focused exercises on:

- **Binary Search Trees** — construction, rotation, and iteration (`BSTRotation`, `BSTIterator`)
- **Hash Tables** — custom hash table implementation (`Hashtable`)
- **Interface Design** — designing clean, extensible Java interfaces (`InterfaceDesign`)
- **Full-Stack Role-Based Application** — a backend/frontend project (`RoleCode`) applying data structures concepts within a larger application context
- **Exception Handling & Edge Cases** — robust program design (`ExceptionalCourseEnrollment`)

## Skills Demonstrated

- Implementing and analyzing core data structures (trees, hash tables, graphs, lists) from scratch
- Algorithmic problem solving and complexity analysis
- Object-oriented design principles in Java
- Applying data structures to model real-world systems and constraints
- Exception handling and defensive programming
